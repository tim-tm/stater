# stater
stater (statistic-er) is a useful tool for getting an overview over pieces of data.

## Getting started

Cloning the repository
```sh
git clone https://github.com/tim-tm/stater && cd stater
```

Configuring stater
```sh
cmake -B build
```

Building stater
```sh
cmake --build build
```
or
```sh
make -C build
```

Running an example
```sh
./build/stater examples/pacman.state
```
